# Smartcart-Clustering-System
## 🚀 Unlocking customer insights with AI-powered clustering  
### Smarter, personalized e-commerce engagement

SmartCart Clustering System applies unsupervised ML to segment customers by demographics, purchase behavior, activity, and loyalty. It uncovers hidden patterns to enable personalized marketing, improve retention, and drive smarter, data‑driven decisions for e‑commerce growth.
